# Getting started

* This project is based on [node.js](https://nodejs.org)
* `npm install -g bower`
* `npm install`
* Add `prod.env` and `test.env` file in the project's root.
  These files contain respectively `export ENVIRONMENT="prod"` and `export ENVIRONMENT="test"`.
* `npm start`


# Testing
